## How to deploy on GitHub Pages

1. Create a new public GitHub repository.
2. Upload all files from this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save and wait 1–3 minutes.
6. Open the GitHub Pages URL shown in the Pages settings.

## How to update content

- Edit `index.html` to update text, experience, awards, links, or sections.
- Edit `style.css` to change colors, spacing, layout, or responsive design.
- Edit `script.js` to update menu behavior, reveal animations, active navigation, or copy-email behavior.
- Replace `assets/Muhammad_Fahd_CV.pdf` with your latest CV using the same file name to keep the download button working.
